## How to Access Files

### On GitHub
1. Visit: https://github.com/Osama-Qonaibe/ROUM-Token
2. Browse folders or download specific files
3. Use the green "Code" button to clone or download

### Command Line
```bash
# Clone the repository
git clone https://github.com/Osama-Qonaibe/ROUM-Token.git

# Navigate to contract
cd ROUM-Token/contracts

# View contract source
cat ROUM.sol

# View ABI
cat ../build/abi.json
```

### Download Specific Files
- Individual files can be downloaded via the GitHub web interface
- Right-click on file and select "Save As"
- Or use raw file URLs:
  - https://raw.githubusercontent.com/Osama-Qonaibe/ROUM-Token/main/contracts/ROUM.sol
  - https://raw.githubusercontent.com/Osama-Qonaibe/ROUM-Token/main/build/abi.json
