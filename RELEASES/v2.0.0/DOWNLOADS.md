## Installation Instructions

### For Wallet Integration

1. Add contract address to your wallet:
   ```
   0x218232b3e7e6214A49922de0954cFc8757F7a504
   ```

2. Add token details:
   - Name: roum token
   - Symbol: ROUM
   - Decimals: 18
   - Logo: https://raw.githubusercontent.com/Osama-Qonaibe/ROUM-Token/main/assets/logos/logo.png

3. Network: Binance Smart Chain (BSC)
   - Chain ID: 56
   - RPC: https://bsc-dataseed1.binance.org:443

### For Development

```bash
# Clone repository
git clone https://github.com/Osama-Qonaibe/ROUM-Token.git
cd ROUM-Token

# Install dependencies
npm install

# Run linter
npm run lint

# View contract
cat contracts/ruom.v2.0.sol
```
