# ROUM Token Logos

## Upload Your Logo Files Here

Place your logo files in this directory with these exact names:

- `logo.png` - Main logo (1024x1024px)
- `logo-small.png` - Icon version (256x256px)  
- `banner.png` - Repository banner (1200x400px)

## File Specifications

### Main Logo (logo.png)
- **Size:** 1024 x 1024 pixels
- **Format:** PNG with transparent background
- **Usage:** README header, documentation, promotional materials

### Small Icon (logo-small.png)
- **Size:** 256 x 256 pixels
- **Format:** PNG with transparent background
- **Usage:** Wallet icons (MetaMask, Trust Wallet), mobile apps

### Banner (banner.png)
- **Size:** 1200 x 400 pixels
- **Format:** PNG or JPG
- **Usage:** GitHub repository header, social media covers

## How to Upload

1. Navigate to this directory on GitHub
2. Click "Add file" > "Upload files"
3. Drag and drop your logo files
4. Commit with message: "Add ROUM token logos"

After upload, the logos will automatically appear in the main README.
