## Logo Guidelines

### Primary Logo
- **File:** `logo.png`
- **Recommended Size:** 512x512px or 1024x1024px
- **Format:** PNG with transparent background
- **Usage:** Main branding, GitHub profile, documentation

### Small Icon
- **File:** `logo-small.png`
- **Size:** 256x256px
- **Format:** PNG with transparent background
- **Usage:** MetaMask, Trust Wallet, mobile apps

### Banner
- **File:** `banner.png`
- **Size:** 1200x400px
- **Format:** PNG or JPG
- **Usage:** GitHub repository header, social media covers

### SVG Version
- **File:** `logo.svg`
- **Format:** Scalable Vector Graphics
- **Usage:** Professional applications, high-resolution printing

## Design Recommendations

### Color Palette
- **Primary:** Heritage colors representing Tel Rumeida
- **Accent:** Gold/Bronze for historical significance
- **Background:** Should work on both light and dark themes

### Symbol Elements
- Consider incorporating:
  - Palestinian heritage motifs
  - Tel Rumeida archaeological symbols
  - Ancient architecture elements
  - Olive tree (symbol of Palestine)
